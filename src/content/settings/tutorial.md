## Como editar el sitio

Este documento es una referencia rapida para editar contenido desde este panel.

---

### Formato de texto (Markdown)

En los campos que muestran un editor con barra de herramientas, puede usar formato Markdown:

| Lo que escribe | Lo que se ve |
|---|---|
| `**texto en negrita**` | **texto en negrita** |
| `*texto en cursiva*` | *texto en cursiva* |
| `[Texto del enlace](https://ejemplo.com)` | Un enlace que se puede hacer clic |
| `# Titulo grande` | Encabezado principal |
| `## Titulo mediano` | Encabezado secundario |
| `- Elemento de lista` | Una lista con puntos |
| `1. Elemento numerado` | Una lista numerada |

---

### Bloques de paginas publicas

Cada pagina publica se compone de **secciones** (bloques) que puede agregar, quitar y reordenar:

| Bloque | Para que sirve |
|---|---|
| **Portada principal** | Imagen de fondo grande con titulo, texto y botones |
| **Encabezado de seccion** | Titulo y texto breve para separar secciones |
| **Barra de beneficios** | Iconos con titulo y descripcion corta |
| **Tarjetas** | Cuadros con icono, texto y enlace |
| **Texto** | Parrafo libre con formato Markdown |
| **Texto con imagen** | Texto al lado de una imagen, con botones opcionales |
| **Galeria** | Conjunto de imagenes con visor ampliado |
| **Lista de documentos** | Archivos PDF u otros enlaces para descargar |
| **Autoridades y profesores** | Director y lista de profesorado |
| **Contacto** | Datos de contacto con formulario opcional |

**Para agregar un bloque**: Haga clic en "Agregar secciones de la pagina" y seleccione el tipo.

**Para reordenar**: Arrastre los bloques hacia arriba o hacia abajo.

**Para eliminar**: Haga clic en el bloque y busque la opcion "Eliminar".

---

### Imagenes

- Use el boton de imagen en la barra de herramientas del editor Markdown
- O en campos dedicados de imagen (Portada, Galeria, Texto con imagen)
- Las imagenes se guardan automaticamente en la carpeta del sitio

---

### Portal de Alumnos (Clases, Tareas, Recursos)

Para publicar contenido en el portal:

1. Seleccione la coleccion (Clases, Tareas o Recursos) en el menu lateral
2. Haga clic en "Nueva Clases" (o Tareas / Recursos)
3. Complete los campos y use el editor Markdown para el contenido
4. Haga clic en "Publicar" para que los alumnos lo vean

**Ejemplo de enlace en una tarea:**
```
Revise el material en [Google Drive](https://drive.google.com/...)
```

---

### Avisos / Anuncios

Los avisos aparecen en la pagina publica de Avisos. Cada aviso tiene:
- Titulo y fecha
- Descripcion breve (se muestra en la tarjeta)
- Imagen opcional (se muestra en la tarjeta)
- Contenido completo con formato Markdown

---

### Consejos

- **Siempre haga clic en "Publicar"** para que los cambios se reflejen en el sitio (tarda aprox. 1 minuto)
- Si comete un error, puede editar de nuevo — los cambios anteriores quedan guardados en el historial
- Los campos marcados con *(opcional)* se pueden dejar vacios
- Si necesita ayuda, contacte al administrador del sitio
